# Support

## Need Help?

If you're experiencing issues with Clay Rush or have questions, please reach out through one of the following channels:

### Report a Bug or Issue

Open an issue on our GitHub repository:

[Open a Support Issue](https://github.com/designspin/clay-rush-site/issues/new)

### Common Questions

**Q: How do I restore my "Remove Ads" purchase?**

Go to the title screen, tap "Remove Ads", then select "Restore Purchases" from the menu.

**Q: How does the daily challenge work?**

A new challenge is generated every day based on the date. Complete it to build your streak. Challenges include hitting a certain number of perfects, reaching a combo target, or clearing specific rounds.

**Q: What is the bonus duck?**

If you destroy every single target in an attempt, a bonus duck will fly across the screen after a short delay. It's worth 3,000–6,000 points depending on how many you've hit in the current round. Inspired by the classic arcade game Hyper Sports!

**Q: How do I access the leaderboard?**

Tap the "Leaderboard" button on the title screen. If you haven't signed into Game Center yet, you'll be prompted to sign in.

**Q: The game isn't running smoothly. What should I do?**

Make sure you're running the latest version of iOS and close any background apps. Clay Rush is optimised for iPhone and iPad.
